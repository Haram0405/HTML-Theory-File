Question 0.
    What is HTML and what is the difference between HTML and HTML5?
Answer:
    <b>i- HTML(Old Version):</b> The basic code used to build websites in the past. It could not play video or audio without using slow extra software like flash player.
    <b>ii- HTML5(Modern Standard):</b> The latest version used today. It has native support to play video and audio directly inside the browser.
